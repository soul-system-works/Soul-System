# AGENTS.md

This repository is the source of the **Soul System** — a meta-level philosophy
and operational framework for AI-assisted research, science, engineering, and
coding work.

**The Soul System composes with other methodologies.** It is a layer that sits
above how you already work (BMAD, TDD, Cursor Rules, your own conventions),
not a replacement for it. Adopt what helps. Ignore what does not.

If your agent tool reads `AGENTS.md` (Cursor, Aider, Codex, Gemini CLI, Jules,
Zed, and others), the authoritative files are:

- `operations/CLAUDE.md` — session-start instructions (the Soul Seed)
- `philosophy/the-soul.md` — the governing philosophy
- `operations/` — conventions and processes

These files are written tool-agnostically. The filename `CLAUDE.md` reflects
the historical origin of the framework, not a tool restriction.
