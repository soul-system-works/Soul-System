## The Specific Proposed Changes

### Change 1 — Frame gate ("Before any solution is proposed")

Append to the existing paragraph:

> When a specification already supplies both levels, this gate is a check —
> verify the two are present and coherent — not a production. When no framing
> exists, producing it is the gate's work. The check is not a rubber stamp: an
> incoherent or too-narrow given frame must still be challenged.

### Change 2 — completion obligation ("Before any work is called complete")

Insert, after "The work must survive contact with what is actually true." and
before "And the Universe is not the task —":

> Local correctness is not global correctness — passing every local test is not
> the same as satisfying the system's global invariants (conservation laws,
> end-to-end balances, the behavior the whole is meant to exhibit). Verify the
> global invariant, not only the parts; this is the difference between
> verification and validation.

### Seed mirror (operations/CLAUDE.md)

- "Before any solution" gate gains a one-line check-vs-production note.
- "Before calling anything complete" gate gains a one-line global-invariant note.
