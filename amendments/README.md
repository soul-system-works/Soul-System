# Amendments

The living record of how the Soul System has evolved.

```
proposed/    ← submitted, under deliberation
accepted/    ← accepted, incorporated into documents
returned/    ← returned, with reason and conditions for resubmission
```

Nothing is ever deleted from this folder. The history of what was proposed,
deliberated, and decided is part of the record the Council needs to do its work.

---

## Naming Convention

```
amendments/proposed/[DOMAIN]-A[NNN]-[brief-description].md
amendments/accepted/[DOMAIN]-A[NNN]-[brief-description].md
amendments/returned/[DOMAIN]-A[NNN]-[brief-description].md
```

Example: `amendments/proposed/THERMAL-A001-partial-domain-coverage-discipline.md`

---

## Process

The amendment process is defined in `operations/council-synthesis.md`.
Every proposed amendment must answer the three Amendment Questions
(Evidence, Necessity, Coherence) before it is accepted.
