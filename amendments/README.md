# Amendments

The living record of how the Soul System has evolved.

```
proposed/    ← submitted, under deliberation
accepted/    ← accepted, incorporated into documents
returned/    ← returned, with reason and conditions for resubmission
```

Nothing is ever deleted from this folder.
The history of what was proposed, deliberated, and decided
is part of the record the Council needs to do its work.

See `CONTRIBUTING.md` for the amendment process and naming conventions.
